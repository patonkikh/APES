# Skill Template

Reference template for authoring APES Engineering Playbooks.

## Usage

1. Copy `skill.md` to `skills/<category>/<skill-name>/skill.md`
2. Replace all `[PLACEHOLDER]` values
3. Remove instructional comments (lines starting with `>`)
4. Add `README.md` with usage example
5. Run review checklist from [docs/SKILL_STANDARD.md](../../docs/SKILL_STANDARD.md)

## Cursor install

```text
skills/<category>/<skill-name>/skill.md  →  ~/.cursor/skills/<skill-name>/SKILL.md
```

Optionally add YAML frontmatter locally for Cursor auto-discovery.
