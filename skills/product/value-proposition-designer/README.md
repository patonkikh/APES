# Value Proposition Designer

Value Proposition Canvas: customer jobs/pains/gains mapped to pain relievers and gain creators with fit assessment.

## Example prompt

```text
Build a Value Proposition Canvas for our B2B analytics persona: data analysts
at mid-size e-commerce companies struggling with manual report generation.
Product: self-serve dashboard with AI insights.
```

## Chain position

`persona-generator` → `problem-statement-builder` → **value-proposition-designer** → `product-vision-builder`
