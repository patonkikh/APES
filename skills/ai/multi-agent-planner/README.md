# Multi-Agent Planner

Design agent roles, orchestration patterns, handoff protocols, and failure recovery for collaborative AI systems.

## Chain position

`ai/ai-solution-architect` → **multi-agent-planner** → `ai/ai-workflow-builder`

## Example use

Input: customer support system requiring triage, research, and response agents with escalation rules.  
Output: orchestration plan with supervisor pattern, three specialist agents, message schema, and timeout recovery.
