## [0.5.0] — 2026-07-01

### Added

- **50 Engineering Playbook Skills** across 6 categories
- Product (15): discovery through delivery + OKR, sprint, value prop, north star
- Architecture (10): C4 diagrams, API design, integration, scalability
- AI (10): solution architect through latency optimizer
- RAG (5): architecture through hybrid search
- Security (5): injection detection through governance
- MCP (5): server/client/tools through best practices
- docs/PUBLISHING.md, catalog.json, scripts/generate_catalog.py
- Skill chain docs: AI, RAG, Security, MCP

### Changed

- Stage 2 (Growth) target of 50 Skills reached
- Current stage: Stage 3 (Scale)
